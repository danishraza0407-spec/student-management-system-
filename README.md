# student-management-system-
A simple Student Management System built in C++ that performs operations like add, view, update and delete student records using file handling.
<br>
This project is a simple Student Management System developed in C programming language. It allows users to manage student records efficiently using file handling.

🚀 Features
➕ Add new student record
📄 View all student records
✏️ Update existing student details
❌ Delete student record
